Leetcode

LeetCode is going to be releasing a 30-Day Challenge for a few months where they will release a programming problem every day starting from April 1st, which will be handpicked from their frequently asked interview questions along with 5 new problems. They also have some nice rewards as incentives, but the condition is that you must solve every problem within 24 hours from the release time. This is a very good challenge and opportunity for me to work on my problem solving skills and have a great time solving problems and discussing them with my friends. See: Jaydip Magan

I'm going to solving these problems everyday using my current knowledge, so they might not be optimal but they'll surely solve the problem efficiently.
